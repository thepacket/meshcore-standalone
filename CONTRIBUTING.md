# Contributing

**meshcore-standalone** is an independent, personal derivative of
[MeshCore](https://github.com/meshcore-dev/MeshCore), maintained by
[@thepacket](https://github.com/thepacket).

## Pull requests are not accepted

This project does **not** accept pull requests — any PR opened here is closed
automatically. This is a deliberate choice to keep the project's provenance and
direction clear; it is unrelated to, and implies nothing about, the upstream
MeshCore project. You are very welcome to **fork** and build your own version
(the MIT license permits it).

## Issues

Bug reports and ideas are welcome via the **Issues** tracker, but there is no
guarantee of a response or that anything will be acted on. Please include your
board, build environment, and steps to reproduce.

## Relationship to upstream MeshCore

This repo tracks `meshcore-dev/MeshCore` as a read-only `upstream` remote so it
can pull in future changes. Nothing is pushed or proposed back upstream.
